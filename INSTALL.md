# Инструкция по установке

## Требования

- Python 3.10 или 3.11 (рекомендуется 3.10 для полной совместимости)
- MetaTrader 5 (для загрузки данных)

## Установка зависимостей

### Базовые зависимости

```bash
pip install -r requirements.txt
```

### Опциональные зависимости

#### pandas-ta (для расширенной поддержки индикаторов)

**Для Python 3.10:**
```bash
pip install pandas-ta==0.3.14b
```

**Для Python 3.11:**
```bash
# Попробуйте установить последнюю версию
pip install pandas-ta

# Или установите конкретную версию, если доступна
pip install pandas-ta==0.4.71b0
```

**Важно:** pandas-ta не является обязательной зависимостью. Если она не установлена, код автоматически переключится на ручной расчет индикаторов. Все функции будут работать, но некоторые индикаторы могут быть рассчитаны немного по-другому.

## Проверка установки

После установки проверьте, что все работает:

```python
python -c "import pandas; import numpy; import sklearn; print('Базовые библиотеки установлены')"
```

Для проверки pandas-ta (если установлена):
```python
python -c "import pandas_ta; print('pandas-ta установлена')"
```

## Решение проблем

### Ошибка при установке pandas-ta

Если pandas-ta не устанавливается:
1. Убедитесь, что используете правильную версию Python (3.10 или 3.11)
2. Попробуйте установить без указания версии: `pip install pandas-ta`
3. Если не получается - не переживайте, код работает и без неё

### Ошибка при установке MetaTrader5

Если MetaTrader5 не устанавливается:
1. Убедитесь, что MetaTrader 5 установлен на вашем компьютере
2. Попробуйте установить вручную: `pip install MetaTrader5`
3. Для работы с данными можно использовать CSV файлы вместо MT5
